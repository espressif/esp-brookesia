# ESP RainMaker 应用 — 用户使用说明（简体中文）

> 英文版：[USER_GUIDE_EN.md](USER_GUIDE_EN.md)

本说明面向在 **ESP-Brookesia** 手机框架上运行的 **ESP RainMaker** 风格智能家居应用（组件名 `brookesia_app_rainmaker`）。界面为 LVGL，与云端 **ESP RainMaker** 账户及家庭下的设备、日程、场景等数据联动（具体能力取决于固件与后端配置）。

## 页面切换（左右滑动）

登录成功进入主界面后，在 **主页、定时、场景、我的** 四个主 Tab 之间切换时，除可点击 **底部导航栏**（从屏幕底部边缘唤起）外，主要方式为 **在页面上左右滑动（水平滑动手势）**。

四个主页面从左到右的固定顺序为：

1. **主页**（设备列表与家庭概览）
2. **定时**（日程 / Schedules）
3. **场景**（Scenes）
4. **我的**（My Profile）

- **向左滑动**：进入右侧下一页（例如：主页 → 定时 → 场景 → 我的；在「我的」上继续向左滑会回到 **主页**，首尾相接）。
- **向右滑动**：进入左侧上一页（例如：主页 → 向右滑会到 **我的**；定时 → 向右滑回到主页）。

设备详情、创建/编辑日程与场景等 **子页面** 通常不在上述四页循环内，需通过界面上的 **返回** 等控件回到上一级。

---

### 1. 登录

- 首次进入应用会看到 **登录** 界面。
- 在 **邮箱**、**密码** 输入框中填写您的 RainMaker 账户凭据；点按输入框会弹出 **屏幕键盘** 便于输入。
- 填写完成后，点按 **登录** 按钮。登录过程中会显示等待提示；成功后自动进入 **主页**，并拉取当前家庭下的设备等信息。
- 若登录失败，会弹出错误提示，请检查网络、账号密码或后端服务状态。
- 登录页可能提供 **语言** 等快捷入口（若有），用于切换界面显示语言。

### 2. 主页

- **标题区**：显示当前家庭或分组名称；部分机型上可通过 **「Home >」** 一类入口选择/切换家庭（若产品已启用该功能）。
- **欢迎与常用**：展示欢迎文案；**刷新** 可用于从云端重新拉取首页与设备列表（刷新时可能有加载提示）。
- **设备卡片网格**：以卡片形式展示当前家庭下的设备。每张卡片通常包含：
  - 设备图标与名称；
  - 部分设备类型支持在卡片上 **快捷开关**（如灯、开关、风扇等，以实际设备能力为准）。
- **进入设备详情**：点按设备卡片（非开关区域）可进入对应 **灯 / 开关 / 风扇** 等详情页，调节亮度、色温、风速等（以设备型号与参数为准）。

### 3. 定时（日程 / Schedules）

- **定时** 页面列出当前家庭下的所有 **日程**（定时任务）。
- **启用 / 停用**：每条日程旁的 **开关** 用于在云端启用或停用该日程；切换后列表与首页相关状态会随云端数据更新。
- **新建**：点按 **添加**（「+」）按钮，按向导填写日程名称、时间、重复规则、要执行的 **动作**（如控制某设备）后 **保存**。
- **编辑**：在日程 **卡片上长按**，进入 **编辑日程** 界面，可修改上述内容后保存。
- **删除**：进入页面上的 **编辑** 模式后，通过卡片上的 **删除** 控件移除对应日程（具体图标与布局以界面为准）。

### 4. 场景（Scenes）

- **场景** 页面列出当前家庭下的 **场景**（一组预设的设备动作组合）。
- **执行场景**：点按场景卡片上的 **激活 / 执行** 按钮（具体文案与图标以界面为准），会向云端下发该场景，从而驱动关联设备动作。
- **新建**：点按 **添加** 按钮，配置场景名称与各设备动作后 **保存**。
- **编辑**：在场景 **卡片上长按**，进入 **编辑场景** 界面。
- **删除**：在 **编辑** 模式下，通过场景卡片上的 **删除** 控件移除场景。

### 5. 我的（My Profile）

- **我的** 页面展示当前登录用户的 **头像占位**（通常为邮箱首字符）与 **邮箱**；标题为「我的」或 **My Profile** 类文案。
- **重要 — 操作方式**：本页 **菜单行、右上角设置、以及退出登录** 等控件均为 **长按** 后才会触发对应功能；**短按 / 轻点不会执行**，这是为了避免与 **左右滑动切换页面** 的手势冲突、减少误触。
  - **语言**：长按 **语言** 行，打开语言选择器，切换界面语言。
  - **通知中心 / 隐私政策 / 使用条款** 等行：长按后目前多为 **说明类弹窗**（占位或后续扩展，以实际固件为准）。
  - **右上角设置图标**：长按打开 **设置** 相关提示（占位功能时显示说明文案）。
  - **退出登录**：长按 **退出登录** 区域，确认执行登出；成功后返回 **登录** 界面。
