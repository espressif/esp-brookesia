# ESP-Brookesia Expression Emote

* [中文版本](./README_CN.md)

## Overview

`brookesia_expression_emote` is an ESP-Brookesia expression management component built on the ESP-Brookesia service framework. It provides emote sets, animation sets, and event message sets for visual AI expression.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/expression/emote.html).
