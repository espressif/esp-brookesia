# ESP-Brookesia Custom Service

* [中文版本](./README_CN.md)

## Overview

`brookesia_service_custom` provides a ready-to-use `CustomService` for the ESP-Brookesia ecosystem. It enables user-defined function and event registration and invocation without creating a dedicated service component.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/service/custom.html).
