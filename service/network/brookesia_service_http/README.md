# ESP-Brookesia HTTP Service

* [中文版本](./README_CN.md)

## Overview

`brookesia_service_http` is the HTTP and HTTPS client service for application-layer network requests.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/service/http.html).
