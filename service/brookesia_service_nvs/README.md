# ESP-Brookesia NVS Service

* [中文版本](./README_CN.md)

## Overview

`brookesia_service_nvs` is an NVS (Non-Volatile Storage) service for the ESP-Brookesia ecosystem. It provides namespace-based key-value storage management.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/service/nvs.html).
