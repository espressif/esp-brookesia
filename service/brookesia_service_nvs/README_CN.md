# ESP-Brookesia NVS Service

* [English Version](./README.md)

## 概述

`brookesia_service_nvs` 是为 ESP-Brookesia 生态系统提供的 NVS（Non-Volatile Storage，非易失性存储）服务，提供基于命名空间的键值存储管理能力。

更多信息请参考 [ESP-Brookesia 编程指南](https://docs.espressif.com/projects/esp-brookesia/zh_CN/latest/service/nvs.html)。
