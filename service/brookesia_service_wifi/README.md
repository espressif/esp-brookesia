# ESP-Brookesia WiFi Service

* [中文版本](./README_CN.md)

## Overview

`brookesia_service_wifi` is a Wi-Fi connection management service for the ESP-Brookesia ecosystem. It provides state machine management, AP scanning, connection management, and SoftAP functionality.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/service/wifi.html).
