# ESP-Brookesia NES Emulation Service

* [中文版本](./README_CN.md)

## Overview

`brookesia_emulation_nes` is the NES emulation service for ESP-Brookesia products.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/service/emulation/nes.html).
