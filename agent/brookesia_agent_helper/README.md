# ESP-Brookesia Agent Helper

* [中文版本](./README_CN.md)

## Overview

`brookesia_agent_helper` is an ESP-Brookesia agent development helper library based on C++20 Concepts and CRTP (Curiously Recurring Template Pattern). It provides type-safe definitions, schemas, and unified calling interfaces for agent developers and users.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/agent/helper/index.html).
