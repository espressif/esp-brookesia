# ESP-Brookesia XiaoZhi Agent

* [中文版本](./README_CN.md)

## Overview

`brookesia_agent_xiaozhi` is a XiaoZhi AI agent implementation for the ESP-Brookesia ecosystem. It provides XiaoZhi API integration and unified agent lifecycle management for ESP-Brookesia devices.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/agent/xiaozhi.html).
