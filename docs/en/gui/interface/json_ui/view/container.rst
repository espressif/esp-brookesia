Container
====================

:link_to_translation:`zh_CN:[中文]`

Overview
--------------------

``container`` is an ordinary container node and can also serve as a top-level template root node.

Related Documents
----------------------------------

- :doc:`index`
- :doc:`../styling/layout`
- :doc:`../interaction/events`

Exclusive Props
------------------------------

``container`` has no dedicated props; it mainly uses ``commonProps``, ``layout``, ``placement``, and ``style``.

Common Events
--------------------------

- ``clicked``
- ``scroll``
- ``gesture``
