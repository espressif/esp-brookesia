Button
====================

:link_to_translation:`zh_CN:[中文]`

Overview
--------------------

``button`` represents a button node. A button has no built-in label; to show text, explicitly add a child ``label``.

Related Documents
----------------------------------

- :doc:`index`
- :doc:`../styling/props/common`
- :doc:`../interaction/events`

Exclusive Props
------------------------------

``button`` has no dedicated props; it expresses behavior mainly through ``commonProps``, ``style``, and ``children``.

Common Events
--------------------------

- ``pressed``
- ``released``
- ``clicked``
- ``longPressed``
- ``longPressedRepeat``
