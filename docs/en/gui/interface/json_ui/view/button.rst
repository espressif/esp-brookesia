Button
====================

:link_to_translation:`zh_CN:[中文]`

.. _gui-interface-json_ui-view-button-sec-01:

Overview
--------------------

``button`` represents a button node. A button has no built-in label; to show text, explicitly add a child ``label``.

.. _gui-interface-json_ui-view-button-sec-02:

Related Documents
----------------------------------

- :doc:`index`
- :doc:`../styling/props/common`
- :doc:`../interaction/events`

.. _gui-interface-json_ui-view-button-sec-03:

Exclusive Props
------------------------------

``button`` has no dedicated props; it expresses behavior mainly through ``commonProps``, ``style``, and ``children``.

.. _gui-interface-json_ui-view-button-sec-04:

Common Events
--------------------------

- ``pressed``
- ``released``
- ``clicked``
- ``longPressed``
- ``longPressedRepeat``
