Wi-Fi Shared Types
==================

:link_to_translation:`zh_CN:[中文]`

Public header: ``#include "brookesia/hal_interface/interfaces/wifi/types.hpp"``

``types.hpp`` contains the shared Wi-Fi action, event, station AP, scan, and SoftAP
data structures used by ``BasicIface``, ``StationIface``, ``SoftApIface``,
and the Wi-Fi service helper aliases.
