Utils Helper
============

:link_to_translation:`zh_CN:[中文]`

- Public header: ``#include "brookesia/service_helper/framework/utils.hpp"``

Overview
--------

The Utils helper provides typed memory and thread profiler controls, cached snapshots, events, and immediate memory
sampling through ``get_memory_snapshot()``.
