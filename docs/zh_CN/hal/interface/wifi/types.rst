Wi-Fi 共享类型
==============

:link_to_translation:`en:[English]`

公共头文件： ``#include "brookesia/hal_interface/interfaces/wifi/types.hpp"``

``types.hpp`` 包含 ``BasicIface``、``StationIface``、``SoftApIface``、
以及 Wi-Fi service helper alias 共同使用的 Wi-Fi action、event、Station AP、
扫描和 SoftAP 数据结构。
