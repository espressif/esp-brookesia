frameView
====================

:link_to_translation:`en:[English]`

.. _gui-interface-json_ui-view-frame_view-sec-01:

概览
--------------------

``frameView`` 表示离屏渲染的帧缓冲控件，把自身子树渲染到一块独立帧缓冲，并可将该缓冲注册为命名输出，供其它消费方读取。
