container
====================

:link_to_translation:`en:[English]`

概览
--------------------

``container`` 是普通容器节点，也可作为顶层模板根节点。

相关文档
--------------------

- :doc:`index`
- :doc:`../styling/layout`
- :doc:`../interaction/events`

专属 props
--------------------

``container`` 当前没有专属 props 域，主要使用 ``commonProps``、``layout``、``placement`` 和 ``style``。

常见事件
--------------------

- ``clicked``
- ``scroll``
- ``gesture``
