button
====================

:link_to_translation:`en:[English]`

.. _gui-interface-json_ui-view-button-sec-01:

概览
--------------------

``button`` 表示按钮节点。按钮当前不内置 label；若需要显示文字，必须显式添加子 ``label``。

.. _gui-interface-json_ui-view-button-sec-03:

专属 props
--------------------

``button`` 当前没有专属 props 域，主要通过 ``commonProps``、``style`` 和 ``children`` 表达行为。

.. _gui-interface-json_ui-view-button-sec-04:

常见事件
--------------------

- ``pressed``
- ``released``
- ``clicked``
- ``longPressed``
- ``longPressedRepeat``
