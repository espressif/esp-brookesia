# ESP-Brookesia JavaScript Runtime Backend

* [中文版本](./README_CN.md)

## Overview

`brookesia_runtime_js` is the JavaScript runtime backend for ESP-Brookesia application packages.

For more information, see the [ESP-Brookesia Programming Guide](https://docs.espressif.com/projects/esp-brookesia/en/latest/runtime/js.html).
